# Loadgun

Учебная HTTP-пушка на Go для проверки поведения сервиса под нагрузкой.

Цель проекта разобраться, как устроен простой load generator:
- как задавать RPS
- как запускать параллельные workers
- как ставить timeout на запросы
- как собирать результаты
- как считать throughput, success/fail, p95/p99 latency

## Архитектура

Пушка состоит из четырёх основных частей:

```text
Генератор нагрузки
        ↓
Очередь задач (jobs)
        ↓
Воркеры выполняют HTTP-запросы
        ↓
Сбор результатов и подсчёт статистики
